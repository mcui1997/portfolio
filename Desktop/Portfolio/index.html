Machine Learning

Malware Detector

A machine learning-based malware detection system using Naive Bayes Algorithm to classify and identify malicious software. The system analyzes file signatures, behavioral patterns, and static features to determine if a file is malicious or benign with high accuracy.

Built using Python and scikit-learn, this detector processes various file types and extracts features such as file size, entropy, API calls, and hex signatures. The Naive Bayes classifier was trained on a dataset of over 10,000 samples achieving 94% accuracy in malware detection.

  • Technologies Python, scikit-learn, pandas
  • Category Machine Learning Security
  • Accuracy 94% Detection Rate

Advanced ML

Metamorphic Malware Detector

An advanced malware detection system specifically designed to identify metamorphic malware using Hidden Markov Model (HMM). Metamorphic malware changes its code structure while maintaining functionality, making it difficult for traditional signature-based detection.

This project implements HMM to model the behavioral patterns and instruction sequences of metamorphic malware families. The system analyzes assembly code patterns and creates probabilistic models to detect variants of the same malware family, achieving superior detection rates against polymorphic threats.

  • Technologies Python, HMM, Assembly Analysis
  • Category Advanced Threat Detection
  • Specialization Metamorphic Malware
